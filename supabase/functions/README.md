# Supabase Edge Functions

Aquí quedarán las funciones seguras para:

- Callback y confirmación de Flow.
- Webhooks de pagos y revalidación.
- Procesos de correo con Resend, si se desacoplan del app server.
- Tareas idempotentes de stock y notificaciones.

La primera etapa deja esta carpeta creada para mantener separadas las integraciones sensibles del frontend.
